# ItemID
Find out the id of all blocks by a simple command.

## Commands
Commands | Description | Default | Aliases | Permissions
--------- | ------------------- | ------- | ---------- | -----------
``/itemid`` | ``See the ID of your Item`` | ``true`` | ``id`` | ``none``
